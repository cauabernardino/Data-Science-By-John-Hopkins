# R Input and Evaluation

The things that are typed at the R prompt are called **expressions**.

The `<-` symbol is the assingment operator.

```r
> x <- 1
> print(x)
[1] 1
> x
[1] 1
> msg <- "hello" 
```

The `#` character indicates a comment. Anything in the right is ignored.
